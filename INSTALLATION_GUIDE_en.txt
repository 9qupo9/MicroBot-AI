🚀 MICROBOT AI - INSTALLATION AND LAUNCH GUIDE
================================================================

📋 SYSTEM REQUIREMENTS
------------------------
- Node.js version 18.0 or higher
- npm version 8.0 or higher (or yarn/pnpm)
- Git for cloning the repository
- Modern browser (Chrome, Firefox, Safari, Edge)

🔧 STEP-BY-STEP INSTALLATION
-----------------------

1. CLONE THE REPOSITORY
   git clone <your-repository-url>
   cd microbot-ai

2. INSTALL DEPENDENCIES
   npm install
   
   OR if you use yarn:
   yarn install
   
   OR if you use pnpm:
   pnpm install

3. CONFIGURE ENVIRONMENT VARIABLES
   - Copy .env.example to .env.local:
     copy .env.example .env.local    (Windows)
     cp .env.example .env.local      (Mac/Linux)
   
   - Open .env.local and fill in the necessary keys:
     OPENROUTER_API_KEY=your_openrouter_key
     NEXT_PUBLIC_WALLET_CONNECT_PROJECT_ID=your_project_id

4. RUN IN DEVELOPMENT MODE
   npm run dev
   
   The application will be available at: http://localhost:3000

5. BUILD FOR PRODUCTION
   npm run build
   npm start

🎨 ANIMATIONS AND VISUAL EFFECTS
---------------------------------

The project uses several technologies to create smooth animations:

1. TAILWIND CSS ANIMATIONS
   - Built-in animations: animate-pulse, animate-bounce, animate-spin
   - Custom animations in globals.css:
     * slideInUp - appearance from bottom
     * slideInLeft - appearance from left  
     * fadeInScale - appearance with scaling
     * float - floating effect
     * shimmer - shimmering effect

2. FRAMER MOTION (installed)
   - Advanced transition animations
   - Gesture and interaction animations
   - Version: ^12.23.12

3. CSS TRANSITIONS
   - Smooth color and size transitions
   - Hover effects
   - Transform animations

4. GLASS MORPHISM EFFECTS
   - Translucent elements with blur
   - backdrop-filter: blur()
   - Modern glass design

🚨 IMPORTANT NOTES
-------------------

1. ANIMATIONS ON MOBILE DEVICES
   - Some hover effects are disabled on touch devices
   - Uses @media (hover: none) for optimization

2. PERFORMANCE
   - Animations optimized for 60 FPS
   - Uses transform instead of changing layout properties
   - will-change applied only when necessary

3. ACCESSIBILITY
   - Animations consider prefers-reduced-motion
   - All interactive elements have focus states

4. BROWSER COMPATIBILITY
   - backdrop-filter supported in modern browsers
   - Fallback for older browsers via regular background

🎯 ANIMATION CUSTOMIZATION
---------------------

All custom animations are located in:
- app/globals.css (CSS animations)
- tailwind.config.ts (Tailwind configuration)
- components/*/animations/ (React animation components)

To change animation speed:
1. Open app/globals.css
2. Find @keyframes sections
3. Change duration in animation properties

To disable animations:
1. Add "reduce-motion" class to body
2. Or use CSS: * { animation: none !important; }
