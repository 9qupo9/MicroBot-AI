# Microbot AI - Web3 AI Ассистент

Децентрализованный AI ассистент, созданный с использованием Next.js, интегрирующий OpenRouter для AI ответов, Linera Wallet для аутентификации и смарт-контракты для хранения данных.

## Возможности

- 🤖 **Интеграция AI**: Несколько моделей AI через OpenRouter (GPT-4, Claude, Llama и т.д.)
- 🔐 **Web3 Аутентификация**: Интеграция Linera Wallet для безопасной аутентификации пользователя
- 📱 **Хранение в смарт-контрактах**: Профили пользователей и данные чатов хранятся в блокчейне
- 🌙 **Поддержка тем**: Светлый и темный режимы с плавными переходами
- 💬 **Чат в реальном времени**: Красивый интерфейс чата с индикаторами набора текста
- 🔒 **Децентрализованный**: Отсутствие центрального сервера для хранения пользовательских данных

## Настройка

1. **Установка зависимостей**
   ```bash
   npm install
   ```

2. **Конфигурация окружения**
   Скопируйте `.env.example` в `.env.local` и настройте:
   ```bash
   cp .env.example .env.local
   ```
   
   Заполните вашу конфигурацию:
   - `NEXT_PUBLIC_OPENROUTER_API_KEY`: Ваш ключ API OpenRouter
   - `NEXT_PUBLIC_USER_PROFILE_CONTRACT`: Адрес развернутого контракта профиля пользователя
   - `NEXT_PUBLIC_CHAT_STORAGE_CONTRACT`: Адрес развернутого контракта хранения чатов

3. **Установка Linera Wallet**
   Пользователям необходимо установить расширение браузера Linera Wallet для подключения.

4. **Развертывание смарт-контрактов**
   Разверните контракты профиля пользователя и хранения чатов в вашей предпочтительной сети блокчейн.

5. **Запуск сервера разработки**
   ```bash
   npm run dev
   ```

## Интеграция смарт-контрактов

### Контракт профиля пользователя
- `createProfile(username, avatar)`: Создать профиль пользователя
- `updateProfile(username, avatar)`: Обновить существующий профиль
- `getProfile(address)`: Получить данные профиля пользователя
- `profileExists(address)`: Проверить существование профиля

### Контракт хранения чатов
- `saveChat(chatId, title, encryptedData)`: Сохранить чат в блокчейн
- `getChat(user, chatId)`: Получить конкретный чат
- `getUserChats(user)`: Получить все ID чатов пользователя
- `deleteChat(chatId)`: Удалить чат из блокчейна

## Функции безопасности

- **Шифрованное хранение**: Данные чата шифруются перед сохранением в блокчейн
- **Аутентификация кошельком**: Безопасная аутентификация через Linera Wallet
- **Децентрализованный**: Отсутствие единой точки отказа для пользовательских данных
- **Конфиденциальность**: Пользователи контролируют свои данные через свой кошелек
