🚀 MICROBOT AI - РУКОВОДСТВО ПО УСТАНОВКЕ И ЗАПУСКУ
================================================================

📋 СИСТЕМНЫЕ ТРЕБОВАНИЯ
------------------------
- Node.js версии 18.0 или выше
- npm версии 8.0 или выше (или yarn/pnpm)
- Git для клонирования репозитория
- Современный браузер (Chrome, Firefox, Safari, Edge)

🔧 ПОШАГОВАЯ УСТАНОВКА
-----------------------

1. КЛОНИРОВАНИЕ РЕПОЗИТОРИЯ
   git clone <ваш-репозиторий-url>
   cd microbot-ai

2. УСТАНОВКА ЗАВИСИМОСТЕЙ
   npm install
   
   ИЛИ если используете yarn:
   yarn install
   
   ИЛИ если используете pnpm:
   pnpm install

3. НАСТРОЙКА ПЕРЕМЕННЫХ ОКРУЖЕНИЯ
   - Скопируйте .env.example в .env.local:
     copy .env.example .env.local    (Windows)
     cp .env.example .env.local      (Mac/Linux)
   
   - Откройте .env.local и заполните необходимые ключи:
     OPENROUTER_API_KEY=ваш_ключ_openrouter
     NEXT_PUBLIC_WALLET_CONNECT_PROJECT_ID=ваш_project_id

4. ЗАПУСК В РЕЖИМЕ РАЗРАБОТКИ
   npm run dev
   
   Приложение будет доступно по адресу: http://localhost:3000

5. СБОРКА ДЛЯ ПРОДАКШЕНА
   npm run build
   npm start

🎨 АНИМАЦИИ И ВИЗУАЛЬНЫЕ ЭФФЕКТЫ
---------------------------------

Проект использует несколько технологий для создания плавных анимаций:

1. TAILWIND CSS ANIMATIONS
   - Встроенные анимации: animate-pulse, animate-bounce, animate-spin
   - Кастомные анимации в globals.css:
     * slideInUp - появление снизу
     * slideInLeft - появление слева  
     * fadeInScale - появление с масштабированием
     * float - плавающий эффект
     * shimmer - эффект мерцания

2. FRAMER MOTION (установлен)
   - Продвинутые анимации переходов
   - Анимации жестов и взаимодействий
   - Версия: ^12.23.12

3. CSS TRANSITIONS
   - Плавные переходы цветов и размеров
   - Hover эффекты
   - Transform анимации

4. GLASS MORPHISM ЭФФЕКТЫ
   - Полупрозрачные элементы с размытием
   - backdrop-filter: blur()
   - Современный стеклянный дизайн

🚨 ВАЖНЫЕ ЗАМЕЧАНИЯ
-------------------

1. АНИМАЦИИ НА МОБИЛЬНЫХ УСТРОЙСТВАХ
   - Некоторые hover эффекты отключены на touch устройствах
   - Используется @media (hover: none) для оптимизации

2. ПРОИЗВОДИТЕЛЬНОСТЬ
   - Анимации оптимизированы для 60 FPS
   - Используется transform вместо изменения layout свойств
   - will-change применяется только при необходимости

3. ДОСТУПНОСТЬ
   - Анимации учитывают prefers-reduced-motion
   - Все интерактивные элементы имеют focus состояния

4. БРАУЗЕРНАЯ СОВМЕСТИМОСТЬ
   - backdrop-filter поддерживается в современных браузерах
   - Fallback для старых браузеров через обычный background

🎯 НАСТРОЙКА АНИМАЦИЙ
---------------------

Все кастомные анимации находятся в:
- app/globals.css (CSS анимации)
- tailwind.config.ts (конфигурация Tailwind)
- components/*/animations/ (React компоненты анимаций)

Для изменения скорости анимаций:
1. Откройте app/globals.css
2. Найдите @keyframes секции
3. Измените duration в animation свойствах

Для отключения анимаций:
1. Добавьте класс "reduce-motion" к body
2. Или используйте CSS: * { animation: none !important; }
