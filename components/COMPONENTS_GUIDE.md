## 🎯 Как использовать этот справочник

### 1. **Поиск компонента**
Используйте Ctrl+F для быстрого поиска нужного компонента по названию.

### 2. **Понимание структуры**
- **Feature-based** - компоненты конкретных функций
- **Logical groups** - компоненты по типу задач  
- **UI Components** - базовые UI элементы
- **Compatibility** - для обратной совместимости

### 3. **Добавление нового компонента**
1. Определите категорию компонента
2. Создайте файл в соответствующей папке
3. Добавьте экспорт в `index.ts` папки
4. Обновите этот справочник

### 4. **Импорт компонентов**
```typescript
// Feature-based
import { ChatInterface } from '@/components/chat';

// Logical groups  
import { Container } from '@/components/layout';

// UI Components
import { Button } from '@/components/ui/interactive';

// Compatibility (старый способ)
import { ChatInterface } from '@/components';
```
